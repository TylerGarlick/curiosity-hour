## MVP Scope
- **Platform:** React Native + Expo (iOS/Android)
- **Hosting:** Vercel (web), Expo EAS (mobile builds)
- **Analytics:** PostHog (free tier)
- **Backend:** Supabase (free tier) for 1v1 room matching
- **Questions:** 100 seeded conversation starters

## Success Metrics
- User can start solo session
- User can create 1v1 room and share code
- User can join 1v1 room with code
- Both users see questions and submit answers
- Answers reveal when both submitted
